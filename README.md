# SFramework
Simple PHP MVC framework.

## Install
```
composer require met_mw/sframework
```

## Fast start
1. Install package (composer require met_mw/sframework) into your project root. 
2. Copy all from the "skeleton" folder into your project root folder.
3. Have fun.
