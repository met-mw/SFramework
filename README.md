# SFramework
Simple PHP MVC-Framework.

## Install

### As library
```
composer require met_mw/sframework
```

### As project skeleton
```
composer create-project met_mw/sskeleton
```

## Fast start
### As library
1. composer require met_mw/sframework
2. Have fun.

### As project skeleton
1. composer create-project met_mw/sskeleton
3. Have fun.
